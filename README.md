## 💡 Why I Built This

Soft Computing is one of the most interesting yet underexplored areas in Computer Science.  
Most students study it from textbooks — full of theory, no interaction.

I wanted to change that.

**Adaptive Logic Lab** was born from one simple idea:  
*"What if you could actually play with these algorithms instead of just reading about them?"*

Every single demo here is based on my own original ideas and real-world scenarios —  
from controlling a washing machine with fuzzy logic, to diagnosing diseases, to solving the Traveling Salesman Problem with a genetic algorithm.

Built entirely using **Vibe Coding on Base44** — powered by ideas, logic, and AI-assisted development. No boilerplate. No templates. Just creativity.

---

## 📖 About

**Adaptive Logic Lab** is an interactive educational platform that brings **Soft Computing** to life through **13 live simulations** organized across 4 major categories.

Instead of passive learning, users can:
- 🎮 Interact with real algorithms
- ⚡ Adjust inputs and see instant results
- 🌍 Understand real-world applications
- 🧠 Build intuition for how intelligent systems think
